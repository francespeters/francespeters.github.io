Project Overview

Developed an image compression tool in Java that applies seam carving techniques to reduce image size while preserving important content. The program calculates the lowest energy seams and compresses images efficiently using data structures like LinkedLists to represent pixels. Designed to meet strict runtime constraints with optimized use of 2D arrays.
