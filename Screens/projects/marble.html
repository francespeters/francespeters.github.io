Project Overview

Developed a Marble Solitaire game in Java, employing GitHub for version control and seamless collaboration. Followed the MVC design pattern to decouple core logic, user interface, and data representation, enabling support for multiple board types and future feature expansion. Designed the program with extensibility in mind, using principles like modularity and abstraction to streamline the integration of new features.


Focused heavily on testing, writing unit tests to validate functionality for various board configurations, edge cases, and invalid user inputs. Collaborated in a pair programming environment to refine algorithms and identify bugs efficiently. Leveraged iterative development, creating tests alongside implementation to ensure all features met requirements. Additionally, reviewed code for consistency with design principles and optimized where necessary to maintain robust performance and readability.
