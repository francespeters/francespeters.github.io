Project Overview

SESH is a mobile application designed by a skateboarder to help other skateboarders discover and explore new skate spots nearby. It was designed using Figma and developed using React Native, JavaScript, Google Maps API, and HTML. SESH features an intuitive user interface and reliable data on geographic skate spots. SESH features an intuitive user interface and reliable data about geographic skate spots.

Future Developments

SESH is actively evolving, with planned features including:
