Project Overview

SESH is a mobile application designed by a skateboarder to help other skateboarders discover and explore new skate spots nearby. It was designed using Figma and developed using React Native, JavaScript, Google Maps API, and HTML. SESH features an intuitive user interface and reliable data on geographic skate spots. SESH features an intuitive user interface and reliable data about geographic skate spots.

This is my first mobile development project, and I independently learned React Native to build it. I also explored the capabilities of the Google Maps API, incorporating features like Places to enhance search functionality.

Future Developments

SESH is actively evolving, with planned features including:

User Profiles – Allowing users to save and favorite skate spots.

Community Contributions – Enabling users to submit new locations.

Scalable Data Management – Transitioning from a JSON file to a structured database for better performance.
